# Nossa História — Lucas & Talita 💛

Um sitezinho fofo sobre a nossa trajetória: fotos, declarações e a história
da gente, com uma pegada de arte em *doodle / scribble* (rabiscos feitos à mão).

Feito com carinho para celebrar o nosso casamento em Manchester.

## Objetivo

- Contar a nossa história de um jeito leve e afetivo.
- Mostrar fotos e pequenas declarações ao longo da jornada.
- Ser **lindo no celular** (mobile-first) e também gostoso no desktop.
- Página única, rolagem suave, carregamento rápido.

## Status

🚧 Em construção. Começando pela documentação e estrutura.

## Stack (proposta)

Site estático — sem backend, sem banco de dados. Simples de hospedar e barato
(ou grátis).

- HTML + CSS + um pouco de JavaScript (vanilla, sem framework pesado).
- Fontes com cara de manuscrito + elementos SVG de doodle.
- Deploy sugerido: **GitHub Pages** (grátis, e a repo já vai estar no GitHub).

> A stack pode mudar conforme a gente decidir — isso é só um ponto de partida.
